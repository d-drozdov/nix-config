## Helpful Notes:

If you ever get a NetworkManager-wait-online.service - Network Manager Wait Online error try the following:
```bash
sudo nixos-rebuild boot --flake .#homelab 
sudo reboot
sudo nixos-rebuild switch --flake .#homelab
```
